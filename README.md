# Kaung Khant Mg Mg - Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Vite, featuring a unique Nothing OS-inspired interface.

## Project Structure

- **Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React, React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Features

- **Unique Interface**: Nothing OS inspired aesthetics
- **Boot Sequence**: Custom startup animation
- **Responsive Design**: Mobile-friendly layout
- **Dark Mode**: Sleek dark theme
- **SEO Optimized**: Meta tags and structured content

## Deployment

To build for production:

```bash
npm run build
```
